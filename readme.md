# Jarvis AI Assistant

## Overview
A Python voice assistant that listens to your commands and performs tasks like opening apps, browsing websites, and telling time/date.

## Features
- Voice commands (hands-free control)
- Opens Google, YouTube, Chrome, Spotify, VS Code, Perplexity
- Tells current time and date
- Searches Wikipedia
- Greets you based on time of day

## Technologies Used
- Python 3
- pyttsx3 (text-to-speech)
- SpeechRecognition (speech-to-text)
- PyAudio (microphone)
